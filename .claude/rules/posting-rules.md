# 投稿ルール（絶対遵守）

- **ハルシネーション禁止**: 最新情報解説型の投稿では出典URLが存在する情報のみ使用
- **絵文字禁止**: 投稿本文に絵文字を一切使用しない
- **ハッシュタグ禁止**: 投稿本文にハッシュタグを一切使用しない
- **500文字以上**: 全投稿を必ず500文字以上にする
- **です・ます調統一**: 「しかし」→「でも」、「したがって」→「だから」のように柔らかい口語に統一
