# マーケティングコピーレビューアースキル

セールスコピー・LP・広告文に対して、マーケティング心理学の観点からレビューを行い改善点を提案してください。

## レビュー手順

1. 対象がファイルの場合: `python scripts/copy_checker.py <ファイルパス>` を実行
2. `resources/psychology_checklist.md` で心理テクニックの使用状況をチェック
3. `resources/sales_writing_knowledge.md` のエッセンスと照合
4. `examples/review_output_example.md` のフォーマットでレポート作成

## レポート構成
1. **全体評価**: 5段階スコアと総評
2. **心理テクニック使用状況**: 各テクニックの使用有無と効果度
3. **パワーワード分析**: 効果的な表現と改善余地
4. **CTA分析**: 行動喚起の数・配置・効果
5. **改善提案**: 具体的な修正案（Before/After形式）
6. **最優先アクション**: 今すぐ改善すべき1点

## 注意点
- ターゲット読者に合った使い方を提案する。心理テクニックの詰め込みはNG
- 改善案は必ず「具体的な文章」で示す。抽象的な指摘はNG
- 誇大表現や景品表示法に抵触しうる表現がないかも確認する
- 改善文はこまの文体（`writing-style-clone/resources/style_guide.md`）に従うこと
