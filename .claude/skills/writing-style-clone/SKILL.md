# Writing Style Clone

`style_cloner` エージェントを実行してください。

## 実行手順

1. ユーザーに分析したいサンプル投稿テキストを貼り付けてもらいます（複数可）
2. `style_cloner` エージェントがそのサンプルを深く分析し、`data/style_guide.md` を更新します

## エージェントが分析する要素
- 文末パターン（`...`・`w`・`ー`・体言止め）
- 接続詞の使い方（「でも」「だから」「いや」「正直」）
- 一人称の使い分け（「僕」「私」「自分」）
- 感情表現・自己開示の手法
- AIっぽさを消すテクニック
- 独特のリズム・文体構造

## 参考リソース（エージェントが自動参照）
- `assets/x_post_sample.md` — 5タイプの投稿文体サンプル
- `references/style_guide.md` — 現在のスタイルガイド（更新対象）
- `data/persona.md` — アカウントペルソナ
