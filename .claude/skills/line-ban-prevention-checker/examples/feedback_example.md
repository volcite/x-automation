# 診断フィードバック例

## 危険度総合判定
🚨 **高リスク（BANの可能性大）**

## 検出されたリスク要因
- **コンテンツリスク**: 「月収100万円」「誰でも簡単に稼げる」といった、ガイドラインで禁止されている「確証のない儲け話」に該当する表現が含まれています。
- **行動リスク**: 全リストに対する一斉送信を予定しており、通報率の上昇により凍結されるリスクが高いです。

## 改善提案
- **表現の変更**:
  - ❌「月収100万円も可能！」 → ⭕「売上アップを目指す仕組みをご紹介」
  - ❌「誰でも簡単に稼げる」 → ⭕「未経験からでも取り組める方法」
- **配信設定の変更**: 一斉送信ではなく、直近1ヶ月以内に開封歴のある「エンゲージの高い読者」のみに絞って配信してください。

## 今後の運用アドバイス
万が一BANされた場合に備え、リストのバックアップを定期的に取得し、メールやSMSなど代替となる連絡手段を確保しておくことを強くお勧めします。
