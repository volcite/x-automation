# n8nワークフローバリデータースキル

n8nワークフローのJSONファイルを検証し、潜在的な問題を検出してください。

## 検証手順

1. 対象がJSONファイルの場合: `python scripts/n8n_validator.py <ファイルパス>` を実行
2. `resources/n8n_best_practices.md` と照合
3. `examples/validation_output_example.md` のフォーマットでレポート作成

## レポート構成
1. **ワークフロー概要**: ノード数、接続数、使用ノードタイプ
2. **構造チェック結果**: 必須フィールド、接続の整合性
3. **セキュリティチェック**: Credential、Webhook設定
4. **改善提案**: エラーハンドリング、命名規則等
5. **総合判定**: 本番利用可能 / 要修正 / 致命的問題あり

## 注意点
- JSON構文エラーがある場合は、まずその修正を提案すること
- ノード間の接続切れは最優先で報告すること
- Credential情報の直接埋め込みはセキュリティリスクとして警告すること
