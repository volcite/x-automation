# コンセプト設計アシスタントスキル

商品やコンテンツの企画について、体系的なコンセプト設計を支援してください。

## 設計手順

1. **ヒアリング**: 何を売るか / 誰に向けるか / なぜあなたが提供するか / 競合はどこか
2. **フレームワーク分析**: `resources/concept_design_framework.md` で3C分析・USP抽出・ポジショニングマップ作成
3. **ターゲット分析**: `resources/target_analysis_template.md` でペルソナ設計
4. **コンセプト案の作成**: `examples/concept_output_example.md` のフォーマットで3案以上を提案

## コンセプト案に含める要素
1. **コンセプト文**: 20文字以内のキャッチコピー
2. **ターゲットペルソナ**: 具体的な1人のイメージ
3. **Before→After**: ターゲットの変化
4. **差別化ポイント**: 競合と何が違うのか
5. **キーメッセージ3つ**: 訴求すべき核心メッセージ
6. **NGメッセージ**: 言ってはいけないこと

## 注意点
- 「みんなに売りたい」は「誰にも売れない」。ターゲットは必ず具体的に絞ること
- コンセプトは「お客様の言葉」で語る。専門用語や業界用語は避ける
- 文章を含める場合は、こまの文体（`writing-style-clone/resources/style_guide.md`）に従うこと
