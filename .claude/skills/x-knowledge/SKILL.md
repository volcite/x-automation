# X Knowledge Stock

オーナーの思想・哲学・体験をナレッジストックに登録するスキルです。

対話的にオーナーから情報を聞き取り、`data/knowledge_stock.json` に追記します。

## 実行手順

1. まず `data/knowledge_stock.json` を読み込んで、現在のストック状況を把握する
2. オーナーに以下を質問する:
   - 「どんな考えや体験を登録しますか？URLの貼り付け、テキストメモ、何でもOKです」
3. オーナーの入力を受け取ったら、以下を自動判定する:
   - **category**: `philosophy`（思想・哲学・持論）/ `experience`（体験・経験）/ `quote`（引用・共感した言葉）
   - **topic**: 短いタイトル（20文字以内）
   - **content**: 投稿素材として使える形に整形（原文のニュアンスを保つ）
   - **tags**: 関連キーワード（3〜5個）
   - **priority**: `high`（核として使いたい）/ `medium`（味付け用）
   - **slot_affinity**: `morning`（教育型向き）/ `evening`（共感型向き）/ `both`
4. 整形結果をオーナーに確認してから、以下のコマンドで登録する:
   ```bash
   bash scripts/pipeline_knowledge.sh '{"topic":"...","content":"...","category":"...","priority":"...","tags":[...],"slot_affinity":"..."}'
   ```
5. 「他にも登録したいものはありますか？」と聞いて、連続登録に対応する
6. 終了時に現在のストック件数を報告する

## 初回棚卸しモード

オーナーが「まとめて登録したい」「棚卸ししたい」と言った場合:
1. 以下のカテゴリごとに質問する:
   - 「普段から大切にしている考え方や哲学はありますか？」（philosophy）
   - 「印象に残っている仕事の体験や、転機になった出来事はありますか？」（experience）
   - 「共感している著名人の言葉やフレーズはありますか？」（quote）
2. 各回答を1件ずつ整形して登録する
3. 目標: 10〜20件のストックを構築する

## 注意事項
- オーナーの言葉をそのまま活かすこと。AIっぽく書き換えない
- 曖昧な入力でも「つまりこういうことですか？」と確認して整形する
- 技術的な知見は登録しない。思想・哲学・体験のみ
