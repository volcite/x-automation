# X Editor

`editor` エージェントを実行してください。

エージェントは `data/draft.json` を以下11点でチェックします：
1. 文字数（500文字以上）
2. フック評価・バズ要素反映
3. 炎上リスク
4. トンマナ・文体
5. AIっぽさ（厳重チェック）
6. スマホ最適化
7. 構成力
8. パターン化回避
9. 目的達成（CTA）
10. フォロワー獲得要素
11. 拡散ポテンシャル（eveningスロットのみ）

承認の場合は `data/approved_post.json` に保存。
差し戻しの場合はフィードバックを報告して終了してください。
