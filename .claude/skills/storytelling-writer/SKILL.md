# Storytelling Writer

`storytelling` エージェントを実行してください。

## 使用タイミング
`data/content_plan.json` の `style_type` が「**共感ストーリー型**」のときに、通常の `writer` エージェントの代わりに使用します。

## エージェントが実践する5つのストーリーテリング原則
1. **ヒーローは「過去の自分」**: 完璧な成功談ではなく、等身大の失敗・葛藤から始める
2. **感情の谷・山の設計**: 痛み（谷）→ 気づき（転換）→ 成長（山）→ 読者への橋渡し
3. **「心の声」を二重カギ括弧で**: 「「自分には無理だ」ってずっと思ってた」
4. **感情トリガー黄金順序**: 共感→好奇心→恐怖→希望→驚き→共感→行動
5. **読者への手渡し**: 「あなたにもできる」で締める
