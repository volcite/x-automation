# コンテンツ品質チェッカースキル

提示された文章に対して品質チェックを行い、改善点を指摘してください。

## チェック手順

1. 対象がファイルの場合: `python scripts/quality_checker.py <ファイルパス>` を実行
2. `resources/quality_checklist.md` の各項目で評価
3. `examples/quality_review_example.md` のフォーマットでレポート作成

## レポート構成
1. **品質スコア**: 各観点の5段階評価
2. **自動検出された問題**: スクリプトで検出した項目
3. **読みやすさの改善**: 文長・文末・段落の改善案
4. **表現の改善**: より伝わる言い回しの提案
5. **修正済みテキスト**: 改善箇所を反映した修正版（任意）

## 注意点
- 文体は「正しさ」ではなく「効果」で評価する。口語体が効果的なら口語でOK
- Note記事とX投稿では基準が異なる。X投稿は短さ・インパクト重視
- こまの文体（`writing-style-clone/resources/style_guide.md`）のルールに沿っているかも確認すること
- 「〜ですね」「...」「体言止め」等はこまの文体の特徴であり、修正対象としないこと
