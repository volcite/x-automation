# コンテンツ品質チェックリスト

## 📝 読みやすさ

- [ ] 一文が80文字以下か（120文字超は要分割）
- [ ] 同じ文末表現が3回以上連続していないか
- [ ] 段落は300文字以内に収まっているか
- [ ] 適度に空行が入っているか（スマホで読みやすいか）
- [ ] 漢字率が20〜30%の範囲に収まっているか

## ✍️ 表記の統一

- [ ] 数字が全角・半角で統一されているか
- [ ] 英字が全角・半角で統一されているか
- [ ] 「！」「？」が全角・半角で統一されているか
- [ ] 固有名詞の表記が統一されているか（例: n8n / N8N / エヌエイトエヌ）

## 🧹 冗長表現

- [ ] 二重表現がないか（「まず最初に」等）
- [ ] 「という」「ということ」が多用されていないか
- [ ] 「〜することができる」→「〜できる」に短縮できないか
- [ ] 不要な修飾語はないか（「非常に」「とても」の乱用）

## 🎯 メッセージ性

- [ ] 冒頭の1〜3行で読者の興味を引けているか
- [ ] 各セクションで「だから何？（So What?）」に答えられているか
- [ ] 読者が「次に何をすべきか」が明確になっているか
- [ ] 抽象的な主張に具体例がついているか

## 📱 プラットフォーム別チェック

### Note記事
- [ ] タイトルが64文字以内か（Note推奨）
- [ ] サムネイル画像の設定を忘れていないか
- [ ] 目次が自動生成される見出し構造になっているか

### X投稿
- [ ] 280文字以内か（長文投稿の場合は適切にスレッド化）
- [ ] 1行目でスクロールを止める力があるか
- [ ] 行間に適切な空白があるか（視覚のリズム）
- [ ] CTAが最後にあるか（RT・フォロー・リンククリック等）
